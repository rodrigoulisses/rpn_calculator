# CLI RPN Calculator
It is a implementation of command-line reverse polish notation calculator using Ruby language.

## Requirements
 - Ruby
 - RSpec

## Installation
 ### Clone this project
 ```shell
 git clone git@github.com:rodrigoulisses/rpn_calculator.git
 ```

 ### Run bundler
 ```shell
 bundle install
 ```

## Runner specs
 ```shell
 rspec spec
 ```

## Runner code
 ```shell
 ruby main.rb
 ```
